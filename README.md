# Jocova Financial — Sales Tools

Private sales tool suite for Zach Taylor, Jocova Financial Inc.

## Tools

| Tool | Description |
|------|-------------|
| `nepq-voice-simulator.html` | Live voice call roleplay with scoring and debrief |
| `sector-brief.html` | AI-generated prospecting briefs for 12 sectors |
| `onboarding-wizard.html` | Post-call vendor intake → CRM summary + follow-up kit |
| `sequence-builder.html` | 3-touch outreach sequences (email + phone + SMS) |

## Setup

Hosted on GitHub Pages (private repo + Pages enabled).

Access at: `https://zacharydtaylor.github.io/jocova-tools/`

## Usage

All tools call the Anthropic API directly from the browser. No backend required.
